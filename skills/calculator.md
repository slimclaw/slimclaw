## Instructions
When the user asks for calculations, use the `calculate` tool to evaluate expressions.
Supports basic arithmetic, exponents, and common math functions.

## Tools
### calculate
Evaluate a mathematical expression.
Parameters:
- expression (string, required): The math expression to evaluate (e.g. "2 + 3 * 4")
